# Live Deepfake Detection and Face Recognition System

A full-stack AI-powered web application built using React and Django for real-time face recognition and deepfake detection.

## Features

* Live Webcam Capture
* Face Recognition using DeepFace
* Multiple User Recognition
* Deepfake Detection Ready
* React Frontend
* Django REST API Backend
* Modern Glassmorphism UI

## Tech Stack

### Frontend

* React.js
* Bootstrap
* Axios
* React Webcam

### Backend

* Django
* Django REST Framework
* DeepFace
* TensorFlow
* OpenCV

## Installation

### Backend

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py runserver
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
